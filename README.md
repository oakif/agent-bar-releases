# AgentBar

Native macOS menu-bar app that shows your Claude Code (and eventually Codex)
usage at a glance.

## Install

Simply drag and drop **AgentBar.app** into your Applications from the DMG.

## Features

- Live usage gauge right in the menu bar
- Popover with a detailed usage breakdown
- Automatic updates

## Requirements

macOS 26 or later
